# Descripción

El presente proyecto consta en desarrollar un agente muy simple encargado de gestionar finanzas personales.
Usando el framework LangGraph el agente dispondrá de herramientas con las cuales podrá:
- interactuar con una base de datos que inicialmente es un archivo .csv

1. Consultar datos.

2. Agregar filas de gastos
3. Generar código para graficar pedidos del usuario usando pandas, matplotlib y seaborn

Como LLM se usará Gemini de Google por los créditos gratis que permite usar de sus modelos frontier.
